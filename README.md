# dotfiles

System setup, tooling & configuration

## Installation

```sh
# Clone this repository
$ git clone git@github.com:jdrydn/dotfiles ~/.dotfiles
$ cd ./.dotfiles

# Install submodules
$ git submodule init
$ git submodule sync --recursive
$ git submodule update --init --recursive --jobs "1"

# Initialise
$ ./update.sh
```
